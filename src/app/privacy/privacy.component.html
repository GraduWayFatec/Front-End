DECLARAÇÃO DE PRIVACIDADE

Esta Declaração de Privacidade explica como o Grupo Jacto coleta e usa dados pessoais e descreve os direitos que o Indivíduo tem com relação aos seus dados pessoais.

O Grupo Jacto entende como sensível e relevante os dados pessoais deixados por você (“Indivíduo”) na utilização dos diversos sites, plataformas, sistemas e aplicativos (“Páginas”), do Grupo Jacto, servindo a presente Declaração de Privacidade (“Declaração”) para regular, de forma simples, transparente e objetiva, quais dados e informações serão obtidos, assim como quando os mesmos poderão ser utilizados. 

1. QUAIS SÃO OS DADOS ABRANGIDOS?

Na presente Declaração, entende-se por “dados pessoais” qualquer informação relativa a um Indivíduo que possa ser identificada, direta ou indiretamente, nomeadamente por referência a um identificador, como um nome, um número de identificação, dados de localização ou um identificador em linha. Os dados pessoais também se referem a um ou mais fatores específicos da identidade física, fisiológica, genética, mental, econômica, cultural ou social de um indivíduo.

Também inclui categorias especiais de dados pessoais (dados de categoria especial) a partir dos quais poder-se-á determinar ou inferir os dados de um Indivíduo.

2. DIREITOS DO TITULAR

Importante ressaltar que o Indivíduo sempre possui direitos relativos a privacidade e à proteção dos seus dados, e o Grupo Jacto, além de se preocupar com a segurança desses dados, também se preocupa que o Indivíduo tenha acesso e conhecimento de todos os seus direitos. Os mesmos estarão disponíveis para o Indivíduo por meio do link Portal de Atendimento de Requisições dos Titulares de Dados, onde o Encarregado pelo Tratamento de dados Pessoais, Sr. Marcio Machado, fará o atendimento em resposta aos titulares de dados via link:

Além disso existe o Centro de Preferências de Consentimento, que permite que qualquer titular de dados possa acessar e atualizar suas preferências de consentimento de uso dos dados. Para isso acesse o link:

Dessa forma, abaixo foram resumidos todos os direitos que o titular de dados possui, com uma breve explicação de cada:

  • Acesso

Este direito permite que o Indivíduo possa requisitar e receber uma cópia dos seus dados pessoais que o Grupo Jacto possui sobre o titular dos dados, bem como requerer a confirmação do tratamento que o Grupo Jacto realiza com os dados pessoais e/ou qualquer outra informação que seja importante para o Indivíduo.

A identidade do Indivíduo precisará ser confirmada antes do titular dos dados, receber acesso aos seus dados pessoais.

Caso o Indivíduo não queira preencher o formulário disponibilizado no canal eletrônico do Grupo Jacto, poderá optar por escrever uma carta registrada, com os seguintes itens:

  • Nome completo;
  • Endereço completo;
  • Número de telefone diurno para contato;
  • Número do CPF;
  • Eventualmente, nomes de arquivos ou tipos específicos de registros aos quais o Indivíduo solicita acesso, incluindo datas específicas desses registros, sempre que possível.

O Indivíduo deve enviar a carta registrada com a sua solicitação para Máquinas Agrícolas Jacto S.A., Rua Doutor Luiz Miranda, nº 1.650, Bairro Pirajá, na cidade de Pompeia, Estado de São Paulo, CEP 17580-000, aos cuidados do Encarregado de Proteção de Dados no Escritório de Privacidade.

Todas as solicitações formais de acesso serão direcionadas ao responsável pela privacidade, que analisará cada solicitação. O Indivíduo será notificado se o acesso aos registros solicitados será concedido ou negado e quais isenções se aplicam. O responsável pela privacidade também receberá e solucionará todas as reclamações de privacidade recebidas pelo Grupo Jacto.

  • Retificação

Este direito permite que o Indivíduo possa solicitar a correção, complementação e/ou retificação dos seus dados pessoais, caso identifique qualquer erro e/ou equivoco neles. Contudo, para ser efetivada essa correção, o Grupo Jacto deverá checar a validade dos dados que o Indivíduo está fornecendo.

Se o Indivíduo acredita que há um erro em seus dados pessoais, o Indivíduo tem o direito de solicitar que as informações sejam corrigidas através Portal de Atendimento de Requisições dos Titulares de Dados.

O Grupo Jacto pode solicitar que o Indivíduo forneça documentação para mostrar onde os arquivos do Grupo Jacto estão incorretos. Caso a solicitação seja procedente, o Grupo Jacto irá alterar os dados incorretos dentro de 30 (trinta) dias e notificará o Indivíduo assim que a correção solicitada for concluída.

O Grupo Jacto notificará qualquer pessoa ou organização para a qual seus dados pessoais foram divulgados dentro de um ano antes de o Indivíduo solicitar a correção e os informará sobre a correção ou declaração de desacordo.

  • Eliminação

Este direito permite que o Indivíduo possa solicitar ao Grupo Jacto a exclusão dos seus dados pessoais dos sistemas do Grupo Jacto. Todos os dados coletados serão excluídos dos servidores quando o Indivíduo assim o requisitar ou quando estes não forem mais necessários ou relevantes para oferecer os serviços prestados pelo Grupo Jacto, salvo se houver qualquer outra razão para a sua manutenção, como eventual obrigação legal de retenção de dados ou necessidade de preservação destes para resguardo de direitos do Grupo Jacto.

O Grupo Jacto não armazena dados pessoais sem uma finalidade predefinida e documentada. São seguidas as leis que exigem a exclusão de dados pessoais se o motivo de sua coleta e armazenamento não existir mais.

Os dados pessoais que o Grupo Jacto mantém são baseados também no consentimento que o Indivíduo forneceu, caso o Indivíduo deseje ser removido dos sistemas do Grupo Jacto antes do período de retenção indicado.

Se o Indivíduo registrou seus dados pessoais com o Grupo Jacto, pode desativar sua conta a qualquer momento. Por motivos de segurança, o Grupo Jacto implementou um período de carência de 15 (quinze) dias depois que o Indivíduo solicita a exclusão da conta; no entanto, fazer login na conta durante o período de carência reativará a conta. Para evitar a representação, uma vez que a conta seja desativada e após o término do período de carência, a conta será irrevogavelmente suspensa.

  • Oposição ao processamento

O Indivíduo também tem o direito de contestar o tratamento dos dados ao questionar onde e em que contexto o Grupo Jacto está tratando seus dados pessoais para diferentes finalidades. Em determinadas situações, pode-se demonstrar que o Grupo Jacto tem motivos legítimos para tratar os dados do Indivíduo, os quais poderão se sobrepor aos direitos do Indivíduo, caso, por exemplo, sejam essenciais para o fornecimento do serviço do Grupo Jacto.

  • Portabilidade de dados

O Grupo Jacto fornecerá ao Indivíduo, ou a terceiros que o Indivíduo escolheu, seus dados pessoais em formato estruturado e a confirmação da transferência para a empresa requerida.

O Indivíduo poderá obter e reutilizar os dados pessoais mantidos pelo Grupo Jacto para seus próprios fins em diferentes serviços. O Grupo Jacto permite mover, copiar ou transferir dados pessoais facilmente de um ambiente de Tecnologia da Informação para outro, de maneira segura e protegida, sem prejudicar a sua usabilidade. Este direito se aplica aos dados pessoais mantidos pelo Grupo Jacto, onde o processamento seja automatizado e usado, levando em consideração a prestação de serviços, dentro do contrato e/ou relação que o Indivíduo possui com o Grupo Jacto ou onde esse processamento foi baseado no consentimento que o Indivíduo forneceu ao Grupo Jacto.

  • Confirmação

Por meio desse direito é possível confirmar se existem dados pessoais do Indivíduo nas bases de dados no Grupo Jacto.

  • Restrição de processamento ou Retirar o seu consentimento

Este direito permite que o Indivíduo solicite suspender o processamento de seus dados pessoais nos seguintes cenários: (a) se quiser que o Grupo Jacto estabeleça a precisão dos dados; (b) quando precisar que sejam mantidos os dados mesmo se o Grupo Jacto não precisar mais deles, conforme necessário, para estabelecer, exercer ou defender reivindicações legais; ou (c) se opor ao uso de seus dados, mas nesta hipótese o Grupo Jacto precisa verificar se há motivos legítimos para usá-los.

O Indivíduo tem o direito de retirar o seu consentimento através canal para Atendimento de Requisições dos Titulares de Dados, no entanto, isso não afetará a legalidade de qualquer processamento realizado anteriormente. Se o Indivíduo retirar o seu consentimento, talvez o Grupo Jacto não consiga fornecer e/ou determinados serviços.

Talvez seja necessário solicitar informações específicas do Indivíduo para ajudar o Grupo Jacto a confirmar sua identidade e garantir seu direito de acessar seus dados pessoais (ou de exercer seus outros direitos). Esta é uma medida de segurança para garantir que os dados pessoais não sejam divulgados a qualquer pessoa que não tenha direito de recebê-los.

3. FINALIDADES PARA AS QUAIS O GRUPO JACTO PROCESSA DADOS PESSOAIS

O Grupo Jacto processa dados pessoais para diversos fins. Esses dados pessoais são coletados diretamente do Indivíduo, por exemplo, se o Indivíduo contratar o Grupo Jacto, se o Indivíduo visitar as Páginas do Grupo Jacto, se o Indivíduo enviar seus dados de contato para receber comunicações de marketing do Grupo Jacto ou enviar uma solicitação de emprego através do site de carreiras do Grupo Jacto. Alternativamente, o Grupo Jacto processa os dados pessoais do Indivíduo no contexto e que se encontra no momento do fornecimento dos dados pessoais pelo Indivíduo. Finalmente, O Grupo Jacto também obtém os dados pessoais do Indivíduo através de fontes publicamente disponíveis, como o LinkedIn e outras mídias sociais. Este aviso de privacidade destina-se a cobrir todos os cenários acima mencionados.
